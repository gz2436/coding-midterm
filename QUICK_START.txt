🎮 如何使用

Landing Page:
• 点击任意卡片进入作品
• 鼠标按住创造粒子

Jellyfish 页面:
• 鼠标移动 - 影响粒子
• 点击 - 切换颜色模式
• 空格键 - 也可以切换颜色

Coral Reef 页面:
• 按住鼠标 - 吸引鱼群
• 点击 - 产生气泡

Ocean Waves 页面:
• 拖动鼠标 - 创造涟漪
• R 键 - 重置波浪

📤 GitHub Pages 部署步骤

1. 在 GitHub 创建新仓库 (比如 "ocean-dreams")

2. 上传这些文件到仓库：
   ✅ index.html
   ✅ jellyfish.html
   ✅ coral.html
   ✅ waves.html
   ✅ about.html
   ✅ PROJECT_DESCRIPTION.md (作业提交文档)
   ✅ README.md (可选)

3. GitHub 仓库设置:
   • 点击 Settings
   • 左侧找到 Pages
   • Source 选择 "main" branch
   • 保存

4. 等待 1-2 分钟，你的网站就上线了！
   URL: https://你的用户名.github.io/仓库名/
