ABOUT THIS PROJECT

Ocean Conservation Through Art

Ocean Dreams is an interactive art portfolio combining creative coding with ocean conservation awareness. Through three unique p5.js sketches, this project explores the beauty and fragility of marine ecosystems.

The ocean covers 71% of Earth's surface and is home to incredible biodiversity. However, it faces unprecedented threats from pollution, climate change, and overfishing. This project aims to inspire appreciation for the ocean through digital art.

The Sketches

1. Jellyfish Particle System

A mesmerizing visualization of 20,000 particles arranged in polar coordinates, creating an ethereal jellyfish-like form. The particles respond to mouse movement and cycle through different color palettes.

Techniques: Polar coordinates, particle systems, mathematical animation

2. Coral Reef Ecosystem

A generative art piece featuring procedurally generated coral structures, swimming fish with autonomous behavior, and rising bubbles. The corals sway gently, and fish can be attracted by the mouse cursor.

Techniques: Procedural generation, autonomous agents, gradient backgrounds

3. Ocean Waves

Multiple layers of sine waves create the rhythmic motion of ocean waves. Users can interact by creating ripples with mouse clicks and dragging. Water particles add dynamic visual interest.

Techniques: Sine wave layering, particle physics, interactive ripples

Thought Process

The concept emerged from a desire to merge technical creativity with environmental awareness. I chose the ocean theme because:

  • Marine ecosystems are visually stunning and diverse
  • Ocean conservation is a critical global issue
  • Fluid dynamics and organic forms create beautiful animations
  • The theme allows for varied creative interpretations

Each sketch explores different aspects of p5.js:

  • Jellyfish: Mathematical precision and particle systems
  • Coral Reef: Object-oriented programming and generative design
  • Waves: Layering and harmonic motion

Challenges Encountered

Performance Optimization

Rendering 20,000 particles at 60fps required careful optimization. Using pixelDensity(1) and efficient draw loops helped maintain smooth performance.

Responsive Design

Creating a seamless experience across desktop and mobile devices required flexible layouts, touch event handling, and adaptive UI elements.

Color Harmony

Balancing vibrant ocean colors while maintaining readability and aesthetic coherence took experimentation with HSB color mode and transparency.

Navigation Structure

Designing an intuitive multi-page portfolio that allows easy exploration while keeping each sketch immersive required careful UX consideration.

Technical Stack

  • p5.js 1.9.0 - Creative coding framework
  • HTML5 & CSS3 - Structure and styling
  • Vanilla JavaScript - No frameworks needed
  • GitHub Pages - Hosting platform
